# 🎮 Custom RPG Game (Python)

Цей проєкт присвячений розробці простої RPG-гри на Python. Мета — створити модульну систему для реалізації базових ігрових механік: персонажі, бої, інвентар та події.

---

## 🧠 Концепція

Гра представляє собою текстову RPG, у якій гравець досліджує світ, бореться з ворогами та розвиває свого персонажа.

---

## ⚙️ Функціонал

- 🧍‍♂️ Система персонажа (HP, атака, рівень)
- ⚔️ Покрокова бойова система
- 🎒 Інвентар (базовий)
- 🎲 Випадкові події
- 🤖 Простий AI ворогів

---

## 🛠️ Технології

- Python 3
- ООП (класи та об'єкти)
- Стандартні бібліотеки:
pygame

---

## 🚀 Подальший розвиток

- Додавання графіки (pygame)
- Система квестів
- Розширений інвентар
- NPC та діалоги
- Збереження гри
